# EVRPTW-D：固定换电与移动换电模式下的电动卡车—无人机协同配送研究

## 项目简介
本仓库用于存放硕士论文相关的部分代码、模型验证文件与实验结果。研究对象为**客户时间窗约束下的电动卡车—无人机协同配送路径规划问题**，重点比较两种补能模式：

- **固定换电模式（BSS, Battery Swapping Station）**
- **移动换电模式（SMBS, Smart/Mobile Battery Swapping）**

仓库中的内容主要包括：
1. 使用 **Gurobi** 完成的小规模模型验证文件；
2. 使用 **Java** 编写的启发式求解算法；
3. 部分算例的求解结果与实验输出文件。

## 文件说明
### 1. `EVRPTW-D-BSS.zip`
固定换电模式相关代码或工程文件。

主要内容通常包括：
- 固定换电模式下的模型构建；
- 对应问题实例的求解实现；
- 启发式算法或相关辅助模块。

### 2. `EVRPTW-D-SMBS.zip`
移动换电模式相关代码或工程文件。

主要内容通常包括：
- 移动换电模式下的模型构建；
- 卡车、无人机与换电车协同决策的求解实现；
- 启发式算法或相关辅助模块。

### 3. `Model-Validation.zip`
模型验证文件。

主要用于：
- 基于 **Gurobi** 对小规模算例进行精确求解；
- 验证固定换电模式与移动换电模式数学模型的正确性；
- 为启发式算法结果提供对照参考。

### 4. `evrptw-d-bss-result.zip`
固定换电模式的部分实验结果文件。

可能包括：
- 各类算例的目标值；
- 路径方案；
- 车辆/无人机使用情况；
- 运行日志或汇总结果。

### 5. `evrptw-d-smbs-result.zip`
移动换电模式的部分实验结果文件。

可能包括：
- 各类算例的目标值；
- 协同路径与补能安排；
- 车辆、无人机及换电资源使用情况；
- 运行日志或汇总结果。

## 研究内容概述
本项目围绕两类换电模式下的电动卡车—无人机协同配送问题展开研究，主要关注以下内容：

- 在客户时间窗约束下建立固定换电模式与移动换电模式的优化模型；
- 设计相应的启发式求解算法，提高中大规模算例的求解效率；
- 利用小规模精确求解与中大规模启发式实验，对两类模式的运行效果进行比较分析。

## 使用说明

4. `evrptw-d-bss-result.zip` 与 `evrptw-d-smbs-result.zip` 可用于查看部分实验结果。
